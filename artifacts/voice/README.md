# Voice-Over Assets

These are final narration assets for the Godspeed hackathon demo video.

- `godspeed-voiceover-brian-slow-documentary-final.mp3` - main demo narration, slower documentary style.
- `godspeed-voiceover-brian-normal-speed-long-pauses-v4-final.mp3` - preferred main demo narration with the Brian voice at normal speaking speed, a word-by-word opening title, and longer pauses.
- `godspeed-acronym-brian-slow-documentary-final.mp3` - separate G.O.D.S.P.E.E.D. acronym explanation.

Raw TTS exports, tests and intermediate audio files are intentionally not tracked.
