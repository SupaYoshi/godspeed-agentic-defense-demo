# Demo Video Script

Target length: 2 minutes.

## 0:00 - 0:15

Godspeed is an agentic defense orchestrator. It turns an urgent security scenario into a governed mission with specialist agents, evidence and approval gates.

## 0:15 - 0:35

The enterprise front door can be Microsoft 365 Copilot or Copilot Studio. The user does not need to know which specialist agent to call. They describe the situation in plain language.

## 0:35 - 1:10

Run the mission. Godspeed selects specialist agents for threat intelligence, exposure, vulnerability prioritisation, log coverage, simulation, patch/change planning and evidence.

## 1:10 - 1:35

Godspeed separates facts, assumptions and unknowns. It creates a premortem and identifies what could go wrong if the organisation rushes into action.

## 1:35 - 1:50

Risky actions are blocked behind approval gates: production scanning, patching, isolation and external communication.

## 1:50 - 2:00

The output is a defense package that now has a dev/test Foundry-connected path and can move toward Microsoft Foundry Agent Service and Agent Framework for production-grade orchestration.
