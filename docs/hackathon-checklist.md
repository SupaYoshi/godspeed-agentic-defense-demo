# Hackathon Checklist

## Required

- Working project or agent.
- Project description.
- Demo video on YouTube or Vimeo.
- Public GitHub repository with source code.
- Architecture diagram showing Microsoft Foundry, Microsoft 365 Copilot and/or GitHub Copilot.
- Team member information and Microsoft Learn usernames if submitting as a team.

## Recommended For This Project

- Keep the demo video under 2 minutes.
- Submit under the Reasoning Agents track.
- Use the local sandbox as the working demo.
- Explain the Microsoft-native path clearly.
- Keep private OpenClaw details out of the public repository.
- Do not include secrets, tokens, customer data, internal URLs or private memory.

## Final Review Before Publishing

- Search for private names, internal domains and secrets.
- Check `.gitignore`.
- Run `npm run check`.
- Run the app locally.
- Confirm README and architecture diagram are present.
- Confirm video URL and repo URL are correct.
